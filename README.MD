# Louis Nguyen Portfolio (GitHub Pages)

Minimal static portfolio optimized for GitHub Pages.

## Structure

- `dist/index.html` — single-page portfolio layout
- `dist/style.css` — all styling

## Local preview

```bash
python3 -m http.server 4173 --directory dist
```

Open `http://127.0.0.1:4173`.

## Deploy to GitHub Pages (gh-pages branch)

```bash
npm install
npm run deploy
```
